# SUMAR_HORAS
## Descripción:
Sumar horas en formato HH:MM:SS.
## Propósito:
Reforzar la utilización de operadores aritméticos básicos +, -, *, / y % (suma, resta, multiplicación, división y mod), realizando un sumador de horas.
