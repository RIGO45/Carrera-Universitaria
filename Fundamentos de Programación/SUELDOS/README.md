# SUELDOS
## Descripción:
Aplicar aumento del 15% si su sueldo es inferior a 1000 y el 12% en caso contrario.
## Propósito:
Reforzar la utilización de las condicionales dobles (if y else). Donde aplicamos una condición inicial (if) y realiza una acción, y en caso de que no se cumpla (else), se realiza otra.
