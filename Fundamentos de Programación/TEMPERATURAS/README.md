# TEMPERATURAS
## Descripción:
Mostrar en pantalla las temperaturas de las 4 semanas de los 12 meses del año.
## Propósito:
Aprender a utilizar matrices de arreglos, utilizamos una matriz de 12 * 4 (12 filas y 4 columnas) para mostrar en pantalla las temperaturas por medio de un for.
